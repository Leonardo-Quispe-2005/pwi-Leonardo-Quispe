Ensinando como usar o git pela primeira vez

Primeiramente temos que criar a conta no git

Nessa opção você colocará seus dados e criará sua conta


Com a conta criada, iremos criar um repositório

Vamos dar um nome ao repositório, dar uma descrição e criar o repositório

Com o repositório criado vamos ir nessa opção e copiar o link dele

depois disso vamos usar o git bash na area de trabalho e dentro dele usaremos os seguintes códigos:

$ git config --global user.name "Fulano de Tal"
$ git config --global user.email fulanodetal@exemplo.br

As frases em itálico, você ira trocar pelo os seus dados.

Logo depois iremos escrever "git clone" e colaremos o código da imagem acima

Iremos dar o comando "ls" para descobrirmos as pastas que temos disponiveis e logo após identificarmos a pasta que queremos, vamos dar um cd e o "nome da pasta"




Como mandamos as alterações para o git?

Primeiro digitamos o comando git status, para vermos as alterações realizadas nos arquivos

Usamos o "git add ." para salvarmos todas as alterações

Logo após de todos estarem adicionados iremos dar o comando "git commit -m "comentario sobre os arquivos adicionados""

Logo após a confirmação, usamos o "git push" para mandar para o gitHub. Ele irá abrir uma janela e é só clicar na parte marcada abaixo

E pronto seu repositório esta atualizado no GitHub!!
